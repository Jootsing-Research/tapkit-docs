The TapKit Python SDK is available on PyPI and can be installed with pip.

## Requirements

- Python 3.9 or later
- An active TapKit account with API key

## Installation

Install from PyPI:

```bash
pip install tapkit
```

Or with a specific version:

```bash
pip install tapkit==1.0.0
```
