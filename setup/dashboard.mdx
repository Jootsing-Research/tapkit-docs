The TapKit dashboard provides a web interface for managing your account and devices.

## Overview

Access the dashboard at [tapkit.ai/dashboard](https://tapkit.ai/dashboard).

## Features

### Device Management

- View connected devices
- Monitor device status
- View screen in real-time

### API Keys

- Create new API keys
- View existing keys
- Revoke compromised keys

### Usage & Billing

- View API usage statistics
- Manage subscription
- View billing history

### Settings

- Organization settings
- User management
- Notification preferences

## Getting Started

1. Sign in at [tapkit.ai](https://tapkit.ai)
2. Navigate to the dashboard
3. Connect your first device
4. Generate an API key
5. Start automating!
