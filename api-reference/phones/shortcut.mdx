---
title: Run Shortcut
description: Execute an iOS Shortcut
api: POST /v1/phones/{phone_id}/shortcut
---

Run an iOS Shortcut by index.

## Request

```bash
curl -X POST https://api.tapkit.ai/v1/phones/{phone_id}/shortcut \
  -H "X-API-Key: joot_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"index": 0}'
```

### Request Body

```json
{
  "index": 0
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `index` | `integer` | Yes | Shortcut position (0-based) |
