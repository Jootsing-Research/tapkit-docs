---
title: Pan
description: Slow scroll gesture
api: POST /v1/phones/{phone_id}/pan
---

Execute a pan (slow scroll) gesture in a direction.

## Request

```bash
curl -X POST https://api.tapkit.ai/v1/phones/{phone_id}/pan \
  -H "X-API-Key: TK_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"x": 500, "y": 1000, "direction": "up", "duration_ms": 500}'
```

### Request Body

```json
{
  "x": 500,
  "y": 1000,
  "direction": "up",
  "duration_ms": 500
}
```

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `x` | `integer` | required | Starting X coordinate |
| `y` | `integer` | required | Starting Y coordinate |
| `direction` | `string` | required | `"up"`, `"down"`, `"left"`, or `"right"` |
| `duration_ms` | `integer` | `500` | Pan duration in milliseconds |

## Use Cases

- Precise scrolling
- Controlled navigation
- Reading while scrolling
